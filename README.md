# CIFAR-10 ResNet-18 Adversarial Attack and Defense

This project implements a compact adversarial robustness experiment:

- Dataset: CIFAR-10
- Model: CIFAR-style ResNet-18
- Attacks: FGSM and PGD under `L_inf`
- Defense: PGD adversarial training
- Outputs: checkpoints, training logs, evaluation metrics, robustness curves, adversarial example visualizations, and confusion matrices

Detailed setup and execution steps are in [EXPERIMENT_STEPS.md](EXPERIMENT_STEPS.md).
